I've been using Tasker for a few years now, and over that time have built up dozens of tasks, some small, some more complex, that improve the stock intelligence of my Android phone. Sadly the xml Tasker spits out isn't the easiest to move about, otherwise I'd show profiles via xml snippets. But for now I'll just detail what I use Tasker for.

Display Off

When the screen is off, disable NFC to help optimise battery

  • Context - Display State Off
  • Task - Nfc Off
  • Exit - (restore previous settings)

Charging

While charging, I may as well turn on NFC, Wifi

  • Context - Power Any
  • Task - Wifi On; Nfc On
  • Exit - (restore previous settings)

Wifi Scan

Rather than rely on location services, I turn Wifi on every hour. If I'm in a known location, like Home, then it'll connect. If not my Wifi Idle profile will turn it back off.

  • Context - Time 0700 to 0500 every 1h
  • Task - Wifi On
  • Exit - none

Wifi Idle

I frequently turn my Wifi on automatically. If however it doesn't connect to a network, after 2 minutes I turn it back off. This is to help save battery.

  • Context - Wifi Not Connected
  • Task - Wait 2mins; Wifi Off if(%WIFII !~R >>> CONNECTION <<<)
  • Exit - (disable restore previous settings)

Location

I have Location Services turned off by default. I only switch them on and GPS when opening certain apps.

  • Requires - Root + Secure Settings
  • Context - App; Google Maps, CityMapper, Weather Timeline
  • Task - GPS On; Location Services On
  • Exit - GPS Off; Location Services Off

Adblock via /etc/hosts

Rather than use AdAway, I run my own script so that I can include custom hosts for my home network.

  • Requires - Root + BusyBox
  • Context - Time 0600; Wifi Connected
  • Task -
    1. Run Shell (select 'use Root'):
      echo "127.0.0.1 localhost" > /sdcard/hosts && /system/xbin/wget -O - https://www.dropbox.com/your.custom.hosts.file >> /sdcard/hosts && /system/xbin/wget -O - https://adaway.org/hosts.txt >> /sdcard/hosts && /system/xbin/wget -O - https://hosts-file.net/ad_servers.txt >> /sdcard/hosts && /system/xbin/wget -O - "https://pgl.yoyo.org/adservers/serverlist.php?hostformat=hosts&showintro=0&mimetype=plaintext" >> /sdcard/hosts
    2. Remount: /system Writeable On
    3. Run Shell (select 'use Root'):
      cp /sdcard/hosts /etc/hosts
    4. Remount: /system Writeable Off
  • Exit - (disable restore previous settings)
